Projeto:
O projeto foi desenvolvido com a ideia de fornescer um banco de dados para um Fundos de Investimento Imobiliário (FII), mais especificamente aqueles que investem diretamente em imóveis físicos, tambem conhecido com FII de Tijolo.

O banco de dados fornece 4 esquemas heterogeneas que se vinculam por via d um contrato de alocação de sala mediante a aluguel, mais uma taxa flutuanta da receita liquida da operação, sendo eles
